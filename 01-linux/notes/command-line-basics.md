# Command Line Basics

## What is a Terminal?

A terminal is an application that allows users to interact with the operating system using commands.

Examples:

- GNOME Terminal
- Windows Terminal
- VS Code Terminal

## What is a Shell?

A shell is a program that interprets user commands.

Examples:

- Bash
- Zsh
- Fish

## Terminal vs Shell

Terminal:
The application window.

Shell:
The program running inside the terminal.

## Command Flow

User → Terminal → Shell → Kernel → Hardware

## Summary

The terminal provides access to the shell, which communicates with the operating system.
