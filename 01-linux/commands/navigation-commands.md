# Linux Navigation Commands

## pwd

Purpose:
Display current working directory.

Example:

pwd

## ls

Purpose:
List files and directories.

Examples:

ls
ls -l
ls -la

## cd

Purpose:
Change directory.

Examples:

cd ..
cd ~
cd /

## mkdir

Purpose:
Create a directory.

Example:

mkdir project

## touch

Purpose:
Create an empty file.

Example:

touch notes.txt

## cp

Purpose:
Copy files.

Example:

cp file1.txt file2.txt

## mv

Purpose:
Move or rename files.

Example:

mv old.txt new.txt

## rm

Purpose:
Delete files.

Example:

rm file.txt

## cat

Purpose:
Display file contents.

Example:

cat notes.txt
