# Linux Filesystem Hierarchy

## /

Root directory of the Linux filesystem.

## /home

Stores personal files of users.

## /root

Home directory of the root user.

## /etc

Stores configuration files.

Examples:

- passwd
- hosts
- ssh configuration

## /var

Stores variable data.

Examples:

- Logs
- Cache
- Mail

## /tmp

Stores temporary files.

Files may be deleted automatically.

## /usr

Stores installed applications and utilities.

Examples:

- /usr/bin
- /usr/lib

## Summary

Understanding the filesystem hierarchy helps administrators locate configuration files, logs, applications, and user data.
