# Linux Architecture

## Overview

Linux follows a layered architecture that enables communication between users and hardware.

## Components

### Kernel

The kernel is the core component of Linux.

Responsibilities:

- Process Management
- Memory Management
- Device Management
- Filesystem Management

### User Space

User space contains applications and programs used by users.

Examples:

- VS Code
- Python
- Chrome
- Bash

### Shell

The shell acts as a command interpreter.

Examples:

- Bash
- Zsh
- Fish

## Command Execution Flow

User
↓
Terminal
↓
Shell
↓
Kernel
↓
Hardware

## Summary

The kernel manages hardware resources while the shell allows users to interact with the operating system.
