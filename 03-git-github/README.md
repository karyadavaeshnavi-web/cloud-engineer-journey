# Day 16 – Git Fundamentals

## Topics Covered

- Git
- Version Control
- Repository
- Git Workflow
- git init
- git status
- git add
- git commit
- git log

## Skills Learned

- Initialize Git repositories
- Track file changes
- Stage files
- Create commits
- View commit history

## Commands Practiced

```bash
git --version
git init
git status
git add .
git commit -m "Initial commit"
git log
```

## Real World Application

Git is used by Cloud Engineers to manage Infrastructure as Code, automation scripts, Kubernetes manifests, Dockerfiles, and CI/CD pipelines while maintaining a complete history of every change.
# Day 17 – Remote Repositories & GitHub

## Topics Covered

- GitHub
- Local Repository
- Remote Repository
- git clone
- git push
- git pull
- git fetch
- git remote -v

## Skills Learned

- Connect local repositories to GitHub
- Synchronize code with remote repositories
- Understand collaboration workflows
- Backup projects using GitHub

## Commands Practiced

```bash
git remote -v
git clone <repository-url>
git push
git pull
git fetch
```

## Real World Application

Remote repositories allow Cloud Engineers to collaborate on Infrastructure as Code, automation scripts, Kubernetes deployments, CI/CD pipelines, and cloud configurations while maintaining a centralized project history.
