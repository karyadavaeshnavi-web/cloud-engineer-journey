# Variables

A variable stores a value that can be used later in a program.

Example:

```python
server_name = "web01"
```

Here, `server_name` is the variable and `"web01"` is the value assigned to it.

---

# Data Types

## Integer

Stores whole numbers.

```python
servers = 25
```

---

## Float

Stores decimal values.

```python
cpu_usage = 74.5
```

---

## String

Stores text.

```python
server = "web01"
```

---

## Boolean

Stores either True or False.

```python
service_running = True
```

---

## List

Stores multiple values in a single variable.

```python
servers = ["web01", "web02", "db01"]
```

---
