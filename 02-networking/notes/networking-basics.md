# Networking Basics

## What is a Network?

A network is a collection of devices connected together to exchange information.

Examples:
- Laptop
- Mobile Phone
- Servers
- Printers

---

## LAN (Local Area Network)

A LAN is a network within a small geographical area.

Examples:
- Home WiFi
- Office Network
- College Computer Lab

Characteristics:
- High speed
- Privately managed
- Limited geographical area

---

## WAN (Wide Area Network)

A WAN is a network that spans a large geographical area.

Examples:
- Internet
- Corporate networks connecting multiple cities

Characteristics:
- Covers large distances
- Connects multiple LANs
- Uses public infrastructure

---

## Internet

The Internet is a network of networks.

Communication Flow:

Laptop → Router → ISP → Internet → Server

---

## ISP (Internet Service Provider)

An ISP provides internet access.

Examples:
- Jio
- Airtel
- ACT
- BSNL

---

## Commands Learned

### hostname

Displays the system hostname.

### ip addr

Displays network interfaces and IP addresses.

### ping

Tests connectivity between systems.

### curl

Retrieves content from a URL and tests application connectivity.
