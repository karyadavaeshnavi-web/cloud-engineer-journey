## Commands Practiced

### hostname

Displays the hostname of the current system.

### ip addr

Displays network interfaces and IP addresses.

### ping

Tests connectivity between devices.

### curl

Retrieves data from a URL and verifies application connectivity.

## Key Concepts Learned

### LAN

A network that operates within a small geographical area such as a home, office, or college.

### WAN

A network that connects multiple LANs across large geographical areas.

### Internet

A global network of interconnected networks.

### OSI Model

A framework that explains how data travels between devices through seven layers:

1. Physical
2. Data Link
3. Network
4. Transport
5. Session
6. Presentation
7. Application

---

## Most Important Layers for Cloud Engineers

### Layer 3 - Network

Used in:

* VPCs
* Subnets
* Routing
* Firewall Rules

### Layer 4 - Transport

Used in:

* TCP
* UDP
* Ports
* Load Balancers

### Layer 7 - Application

Used in:

* HTTP
* HTTPS
* DNS
* APIs
